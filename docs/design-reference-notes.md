# Design reference notes

Updated: 2026-07-13

No external site was copied. The upgrade keeps the existing restrained evidence-led visual system
and applies these portable portfolio principles:

1. Make the person's name and cross-track positioning the first identity signal; put role-specific
   entrances immediately below rather than forcing one job title.
2. Treat project pages as working review surfaces, not slide decks: controls, evidence metadata,
   limitations, and source links stay adjacent to the claim they qualify.
3. Use dense tables and framed tools for operational workflows; avoid decorative cards and large
   empty hero areas.
4. Give failures, residuals, and blocked states equal visual legitimacy with successes. Color is
   reinforced by labels and icons.
5. Preserve bilingual navigation and content without changing information architecture between
   languages.
6. On mobile, replace wide diagrams with ordered steps and stack decision panels without removing
   evidence or disclosures.
7. Keep contact paths compact and verifiable. Missing public artifacts are explicit pending states,
   never guessed links.
